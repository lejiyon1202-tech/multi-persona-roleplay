## 핵심 컨셉 변경 요약

| 구분 | v1 (잘못) | v2 (정확) |
|------|----------|----------|
| 학습자 역할 | 시나리오마다 고정 (본부장/그룹장) | **학습자가 직접 선택** |
| 대화 방식 | 학습자가 AI 캐릭터 1명 선택 → 1:1 채팅 | 학습자가 캐릭터 역할을 맡고 → **대화 상대 선택** (1명 또는 다중) |
| 캐릭터 정보 | AI 페르소나용만 | **AI 페르소나용 + 학습자 연기용 양쪽** |

---

## 새 UI 흐름

```
[시나리오 선택]
    ↓
[Step 1: 내가 연기할 캐릭터 선택]
    → 시나리오 내 모든 캐릭터 표시 (학습자가 누가 될지 선택)
    → 각 캐릭터: 이름·역할·배경·가치관·말투·미션·관계도 등 "연기용 detail"
    ↓
[Step 2: 대화 상대 캐릭터 선택 (다중 가능)]
    → 학습자가 선택한 캐릭터를 제외한 나머지 캐릭터 표시
    → 1명 또는 여러 명 선택 가능
    ↓
[채팅 시작]
    → 학습자 = 선택한 캐릭터 역할 연기
    → AI = 선택한 상대 캐릭터(들) 역할 연기
```

## Step 1: 학습자 역할 선택 화면 (character-select.html → step1)

### 핵심 변경
- "대화할 인물을 선택하세요" → **"당신이 연기할 캐릭터를 선택하세요"**
- 모든 캐릭터 선택 가능 (컨텍스트/선택 불가 구분 제거)
- 카드에 "연기용 detail" 표시:
  - 역할 뱃지 + 이름
  - 핵심 가치관 1줄
  - 처한 상황 (학습자 관점)
  - "이 역할 맡기" 버튼 (선택 시 Step 2로)

### 카드 구성 (학습자 연기용)
```
┌──────────────────────────────┐
│ [역할 뱃지]         NO. 01   │
│ 이름 (직책)                  │
│                              │
│ 핵심 가치관: "..."            │
│                              │
│ 당신의 상황:                 │
│ (학습자 관점 상황 1~2줄)     │
│                              │
│ 당신의 미션:                 │
│ (이 캐릭터로서 달성해야 할 것)│
│                              │
│ [이 역할 맡기 →]            │
└──────────────────────────────┘
```

### 클릭 시 상세 정보 (모달)
- 배경 (경력·전공·이전 사건)
- 핵심 가치관 + 말투·언어 특성
- 직면한 압박·심리적 갈등
- 다른 캐릭터와의 관계도
- 추천 발화 패턴·감정 상태

## Step 2: 대화 상대 선택 화면 (partner-select.html 신규)

### 핵심
- 학습자가 선택한 캐릭터 **제외** 나머지 캐릭터 표시
- 다중 선택 가능 (체크박스 방식)
- 선택 수 제한: 1~3명 권장 (합의서에서 상한선 결정)
- "대화 시작" 버튼 활성화 조건: 최소 1명 선택

### 카드 구성 (AI 페르소나 뷰)
```
┌──────────────────────────────┐
│ [역할 뱃지]         [체크박스]│
│ 이름 (직책)                  │
│                              │
│ 핵심 마인드: "..."            │
│ 상황 요약 1줄                 │
│                              │
│ ─────────────────────────   │
│ 이 캐릭터가 어려운 이유:      │
│ (선택 동기 1줄)              │
└──────────────────────────────┘
```

### 선택 확인 바 (하단)
```
[학습자 역할] 강민경 → [상대] 이동현 + 박기훈  [대화 시작 →]
```

## DB/API 변경 요구사항 (카리나 영역)

| 항목 | 내용 |
|------|------|
| sessions 테이블 | `learner_character_id` (학습자 역할 캐릭터) + `dialogue_partner_ids JSON` (상대 캐릭터 배열) |
| scenario_characters | `learner_detail JSON` 신규 필드 (학습자 연기용: 배경·가치관·말투·압박·관계도·발화패턴) |
| API | `GET /api/scenarios/:id/characters` 응답에 `learner_detail` 포함 |
| 세션 생성 | `POST /api/sessions` body에 `learner_character_id` + `partner_character_ids[]` |
